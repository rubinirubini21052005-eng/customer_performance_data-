# Company Performance Analysis

## Project Description
- Analyze company employee performance, department-wise sales, and productivity trends.
- Extract actionable insights for business decisions.

## Tools Used
- Python, Pandas, Matplotlib / Seaborn, Jupyter Notebook

## Dataset
- 20 employees with columns: employee_id, employee_name, department, designation, location, joining_date, monthly_salary, sales_amount, month, performance_rating

## Steps Performed
- Load CSV data into Python
- Check data types & missing values
- Aggregate sales per department & month
- Analyze:
  - Department-wise total sales
  - Top performing employees
  - Average salary by department
  - Performance rating distribution
- Visualize results:
  - Bar charts (department-wise sales)
  - Scatter plots (monthly sales & performance)

## Key Insights
- Sales department generates highest revenue
- Senior sales executives outperform junior staff
- HR & IT departments not directly contributing to sales
- Higher performance ratings correlate with higher sales

## Business Recommendations
- Reward top-performing employees
- Train low-performing staff
- Align marketing & sales strategies
- Use performance ratings for promotions & incentives
